Canon EOS R3

詳細規格
推出日期 約 2021 年 9 月
感光元件像素 2700 萬像素
感光元件大小 36 x 24 mm
感光元件種類 CMOS
鏡頭接環 Canon RF 接環
自動對焦情況 4,779 點（當使用方向鍵選擇自動對焦點時的數量。如自動對焦範圍設定為自動選擇，對焦範圍會劃分為最多 1,053 區）臉部＋追蹤自動對焦（此模式支援眼睛偵測自動對對）單點自動對焦自動對焦擴展（上、下、左及右）擴展自動對焦（周邊）區域自動對焦大區域自動對焦（垂直）大區域自動對焦（水平）人眼及動物眼睛（貓、狗、鳥類）自動對焦主體檢測自動對焦（賽車、電單車）
最大解析度 6000 x 4000
影片解析度 6K RAW (6000 x 3164)：59.94 fps / 29.97 fps / 24.00 fps / 23.98 fps (NTSC) 6K RAW (6000 x 3164)：50.00 fps / 25.00 fps / 24.00 fps (PAL) 4K DCI (4096 x 2160)：119.88 fps / 59.94 fps / 29.97 fps / 24.00 fps / 23.98 fps (NTSC) 4K DCI (4096 x 2160)：100.00 fps / 50.00 fps / 25.00 fps / 24.00 fps (PAL) 4K UHD (3840 x 2160)：119.88 fps / 59.94 fps / 29.97 fps / 24.00 fps / 23.98 fps (NTSC) 4K UHD (3840 x 2160)：100.00 fps / 50.00 fps / 25.00 fps / 24.00 fps (PAL) Full HD (1920 x 1080)：119.88 fps / 59.94 fps / 29.97 fps / 24.00 fps / 23.98 fps (NTSC) Full HD (1920 x 1080)：100.00 fps / 50.00 fps / 25.00 fps / 24.00 fps (PAL)
影片種類 H.265/HEVC（Canon Log 3 開，或 HDR PQ 開）；H.264/MPEG-4 AVC（Canon Log 3 關、HDR PQ 關）分別可選 ALL-I（I-Only）、IPB 兩種
曝光模式
  • 1. 支援光圈先決
  • 2. 支援快門先決
  • 3. 支援手動曝光
    • 場景模式 未有資料
    測光模式 以影像感應器實時測光，384 區（24 x 16）測光拍攝靜止影像：權衡式測光（連結所有自動對焦點）、局部測光 (覆蓋畫面中央約 5.9% 區域)、重點測光 (覆蓋畫面中央約 2.9% 區域)、中央偏重平均測光
    曝光補償 -3EV 至 +3EV 以 1/3EV 或 1/2EV 級調整
    白平衡設定 自動（氣氛優先）、自動（白色優先）、預設（日光、陰影、陰天、鎢絲燈、白光管、閃光燈）、色溫設定（2500 – 10000K）、自訂支援手動白平衡設定
    連拍 30fps 高速連續拍攝（電子快門）、12fps（機械快門）可連續拍攝 410 張或更多的 JPEG 或約 150 張 RAW
    快門速度 1/64000 至 30 秒，支援 B 快門
    ISO 感光值 ISO 100 - 102400、可擴展至 ISO 50-204800
    LCD螢幕 3.2 吋 (415 萬 像素) 側反式多角度觸控屏幕，張開角度：約 175 度、旋轉角度：約 270 度 熒光幕 ，視野率為 100 %
    儲存介面 SD / SDHC / SDXC（兼容 UHS-I、UHS-II）、CFexpress（Type B）
    電池種類 LP-E19 鋰離子充電池
    電池壽命 760 張
    機身重量 nan
    重量(含電池) 1015 克
    體積 150.0 x 142.6 x 87.2 mm
    其他 支援以 USB 電源轉接器 PD-E1（另購）經 USB Type-C 端子以交流電充電。
